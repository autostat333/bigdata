Авторский проект
Проект является авторским, полностью создан одним человеком как результат многолетней работы, проб, ошибок. Одну из ключевых причин вывода проекта в просторы Интернета можно сформулировать как «отдать долг интернету», ибо часть знаний пришли из Интернета — в Интернет должны уйти для других.
